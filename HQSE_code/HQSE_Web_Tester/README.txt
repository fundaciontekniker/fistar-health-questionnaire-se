This folder contains the Health Questionnaire SE Tester

The aim of this Tester is to test the Health Questionnaire SE basic functionalities: GET questionnaire and POST questionnaire

It can be published at Apache Web Server or Apache Tomcat 

Please configure the file js\questionnaire\services-configuration file properly setting the value of QSE_WS_HOST with the ip or dns name of your machine
